# Projects

## Software

[Guitar TUI](https://github.com/konoerik/guitar-tui) | A terminal UI guitar theory application written in Python. The UI uses Textual, the lesson material is written in Markdown. The aim is to provide a distraction free environment for learning and practice.

[Lingo Cards](https://konoerik.github.io/lingocards/) | A progressive web app flash card game for learning other languages. The current vocabulary is suitable for 2-5 year olds. The inspiration was my own son expressing interest in learning the Greek alphabet.

[Prompted Wisdom](https://konoerik.github.io/prompted-wisdom/) | An experiment in AI-generated essays around wisdom and philosophical ideas. Basically using frontier models as talking libraries to gather insights throughout the centuries.

## Experiments

[Exploring Open Source & DRM-Free](open-source-drm-free/) | An ongoing practical experiment in stepping outside the Apple ecosystem, using open source software and DRM-free media.
