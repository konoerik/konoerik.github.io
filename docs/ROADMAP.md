# Roadmap

## Goal
A minimal, fast personal site that reflects who I am — landing, about, projects, and a blog.
Built by hand, owned completely, no moving parts to maintain.

## Phases

### Phase 1: Foundation
Core pages live (landing, about, projects placeholder), blog infrastructure working,
first post published. Deploy pipeline green.

### Phase 2: Content
Fill in projects with real work. Build up blog over time. Refine design.

### Phase 3: Evaluate and trim
Revisit claudify conventions — strip scaffolding that isn't earning its weight for a
project this size. See memory note on this.

## Out of Scope
JavaScript frameworks, CSS frameworks, CMS, comments system, analytics (unless self-hosted and minimal).
